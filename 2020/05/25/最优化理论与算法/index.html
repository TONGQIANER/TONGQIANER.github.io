最优化理论与算法

2020-05-25 Comments

title：作业二

单纯形法

一、单纯形法目标

  1. 原因：在求基本可行解时发现求出的初始基本可行解与图解法求出的最优解不同.

  2. 若线性规划（标准形式）有最优解，则必存在最优基本可行解.因此求解线性规划问题归结为找最优基本可行解.

  3. 单纯形法基本思想：从一个基本可行解出发，求一个使目标函数值有所改善的基本可行解；通过不断改进基本可行解，力图达到最优基本可行解.

二、单纯形法计算步骤

以极小化问题为例. 首先给定一个初始基本可行解. 设初始基为 $B$，然后执行下列步骤：

  1. 解$Bx_B = b$,求得 $x_B = B^{-1}b = \overline{b}$, 令非基变量 $x_N = 0$, 计算目标值 $f = c_Bx_B$.

  2. 求单纯形乘子 $w$, 解 $wB = c_B$, 得到 $w = c_BB^{-1}$. 对于所有非基变量，计算判别数 $z_j-c_j = wp_j-c_j$, 令 $$z_k-c_k = max_{J\in{R}}\left{z_{ij}-c_{j}\right}$$ 若 $z_{k}-c_{k}\leq 0$, 则对于所有非基变量 $z_{j}-c_{j} \leq 0$, 对应基变量的判别数总是零, 因此停止计算，现行基本可行解是最优解. 否则，进行下一步.

  3. 解 $By_{k} = p_{k}$, 求得 $y_{k} = B^{-1}p_{k}$, 若 $y_{k}\leq 0$, 即 $y_{k}$ 的每个分量均非正数，则停止计算，问题不存在有限最优解. 否则，进行下一步.

  4. 确定下标 $r$, 使 $$\frac{\overline{b}{r}}{y{rk}} = \min \left{\frac{\overline{b}{i}}{y{ik}}|y_{ik} > 0\right}$$ $x_{B_{r}}$ 为离基变量,$x_{k}$ 为进基变量. 用 $p_{k}$ 代替 $p_{B_{r}}$, 得到新的基矩阵 $B$.

  5. 检验数最大值入基；根据 $\frac{\overline{b}{i}}{y{ik}}$ 最小值出基；判别数全为负值时为最优解。

三、练习

  1. 用单纯形法解下列线性规划 $$\min Z = 4x_{1} + x_{2}$$$$s.t. 2x_{1} + 3x_{2} \leq 4$$ $$x_{1} + x_{2} \leq 1$$ $$4x_{1} + x_{2} \leq 2$$ $$x_{1},x_{2} \ge 0$$ 解：引入松弛变量 $x_{3},x_{4},x_{5}$ 将上式化为标准形式：$$\min Z = 4x_{1} + x_{2}$$$$s.t. 2x_{1} + 3x_{2} + x_{3} = 4$$ $$x_{1} + x_{2} + x_{4} = 1$$ $$4x_{1} + x_{2} + x_{5} = 2$$ $$x_{j} \ge 0,j = 1…5$$ 系数矩阵：$$A = (p_{1},p_{2},p_{3},p_{4},p_{5}) = \begin{pmatrix}
    2 & 3 & 1 & 0 & 0 \\
1 & 1 & 0 & 1 & 0 \\
4 & 1 & 0 & 0 & 1
    \end{pmatrix}$$ 第一次迭代：
    $$\begin{align}
    B = (p_{3},p_{4},p_{5}) = \begin{pmatrix}
    1 & 0 & 0 \\
0 & 1 & 0 \\
0 & 0 & 1
    \end{pmatrix}\
    B^{-1} = \begin{pmatrix}
    1 & 0 & 0 \\
0 & 1 & 0 \\
0 & 0 & 1
    \end{pmatrix}\
    x^{1} = \begin{pmatrix}
    x_{B}\\
x_{N}
    \end{pmatrix}\
    x_{B} = B^{-1}b = \begin{pmatrix}
    x_{3}\\
x_{4} \\
x_{5}
    \end{pmatrix} = \begin{pmatrix}
    4 \\
1 \\
2
    \end{pmatrix} \
    x_{N} = \begin{pmatrix}
    x_{1}\\
x_{2}
    \end{pmatrix} = \begin{pmatrix}
    0 \\
0
    \end{pmatrix}\
    f_{1} = c_{B}x_{B} = (0,0,0)(4,12,3)^{T} = 0
    \end{align}$$
